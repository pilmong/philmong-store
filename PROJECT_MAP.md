# 🎯 필몽 시스템 통합 나침반 (Project Philmong Roadmap)

이 문서는 필몽 시스템의 두 핵심 애플리케이션의 정체성과 역할을 정의하며, 개발 방향을 안내하는 나침반 역할을 합니다.

---

## 🛍️ 1. 필몽 스토어 (Sales & CRM Portal) - **[현재 앱]**
**"외부 비즈니스의 얼굴이자 베테랑 장사꾼"**

### 정체성 (Identity)
고객과 접점이 있는 모든 판매 활동을 전담하는 비즈니스 포털입니다. **"어떻게 더 잘 팔고, 어떻게 더 정확하게 배송할 것인가"**에 집중합니다.

### 핵심 기능 (Focus)
1. **통합 주문 관리 (OMS)**: 필몽 앱에서 생산된 제품을 가져와 판매. 퀵 에드(텍스트 파싱), 수기 주문 등록 포함.
2. **배송/물류 제어 (TMS)**: 배달 구역 설정, 라이더 배정, 배송 송장 출력.
3. **고객 관계 관리 (CRM)**: B2B 고객사(Clients) 관리, B2C 회원(Users) 관리, 쿠폰 및 프로모션.
4. **판매 채널 최적화**: 채널별 판매가 설정, 영업 현황 대시보드.

### ♻️ 필몽 앱으로 이관/참조할 기능
- **메뉴 기획/식단 편성**: 필몽 앱에서 결정된 식단을 불러와서 전시만 함.
- **상품 마스터**: 필몽 앱에서 상품의 본질(레시피 등)을 정의하고, 여기서는 '판매 설정'만 수행.

---

## 🏛️ 2. 필몽 앱 (Internal Management App)
**"우리 가게의 살림꾼이자 대뇌"**

### 정체성 (Identity)
가게 내부의 운영 효율성을 극대화하기 위한 ERP 시스템입니다. 생산과 매입, 원가에 집중합니다.

### 핵심 기능 (Focus)
1. **생산 마스터**: 제품 정의, 조리/소분 지시서 발행.
2. **구매/재고 관리**: 식재료 발주(Purchase), 거래처(Vendor) 관리.
3. **메뉴 기획**: 시스템의 코어가 되는 식단표 구성.
4. **원가/손익 분석**: 가게의 실질 수익성 계산.

---

## 🚀 필몽 스토어 개편 우선순위
1. [ ] **대시보드 개선**: 판매 현황, 미처리 주문, 배송 현황 등 '장사꾼'에게 필요한 지표 중심 구성.
2. [ ] **메뉴 정리**: 필몽 앱과 중복되는 메뉴(생산 관련)를 숨기거나 '참조' 형태로 변경.
3. [ ] **퀵 에드 & 수기 등록 통합**: 필몽 앱에 있던 강력한 주문 입력 도구들을 이곳으로 완벽 이수.
